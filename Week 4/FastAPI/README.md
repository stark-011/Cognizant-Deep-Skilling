# FastAPI Exercises

This folder contains Week 4 FastAPI practice files covering:

- Basic FastAPI application setup
- Route creation
- Path and query parameters
- Pydantic models
- Dependency injection
- CRUD endpoints
- Response models
- Exception handling
- Middleware
- CORS configuration

Install dependencies with:

```bash
pip install -r requirements.txt
```

Run any example with:

```bash
uvicorn filename:app --reload
```
